If you're new — start here

No idea what a website, Git, or hosting is? That's fine.

Plain English explanations of every concept you need before you start. Skip this if you already know the basics.

🌐

What even is a website?

A website is a set of files (text, images, code) that live on a computer somewhere in the world. When someone types your address into a browser, their computer downloads those files and shows them on screen. That's it.

☁️

What is hosting?

Hosting means renting space on a computer that stays on 24/7 so your website files are always accessible. We use Cloudflare Pages and GitHub Pages — both are completely free and handle everything automatically.

📦

What is Git / GitHub?

Git is a system that tracks changes to your files — like "Save History" on steroids. GitHub is a website where you store those files online. Think of it as Google Drive, but for code. We use it to publish your site for free.

🤖

What is Claude Code?

Claude Code is an AI assistant that lives in your terminal (a text-based window on your computer). You type in plain English — "push this to GitHub" — and it does the technical work for you. No coding knowledge needed.

🎨

What is Clo Design?

Clo Design is an AI-powered website builder. You describe what you want in plain English, and it builds an animated, professional website. You can then export it as files and host it for free. No design skills needed.

⚡

What's the actual flow?

Build a site with Clo Design → export the files → open Claude Code → say "push this to GitHub" → it does it automatically → your site is live. Total time from zero to live: about 2 hours on your first try.

The absolute beginner path
1

Install Claude Code (5 min)

Install Node.js from nodejs.org, then open Terminal on Mac (press Cmd+Space, type "Terminal") and paste: npm install -g @anthropic/claude-code — hit Enter. Done.

2

Create a free GitHub account

Go to github.com → Sign up → free account. This is where your website files will live and be published from. Takes 2 minutes.

3

Build your site with Clo Design

Go to clo.ai/design → describe your site → use prompts from prompts/website-prompts.md → export as code (ZIP). You now have a folder of website files on your computer.

4

Tell Claude Code to push it

Open Terminal in your site folder, type claude to start Claude Code, then say: "Push this folder to a new GitHub repo called my-site". It handles Git, the repo creation, and the push automatically.

5

Your site is live

GitHub Pages activates automatically. Your site is now live at yourusername.github.io/your-site — shareable, free, and online 24/7. For a custom domain, see the Cloudflare guide.

💡 Cold calling tips that actually work

Build a mockup first Spend 30 mins making a demo of their site before you call. Saying "I've already started a mockup" converts 3x better than "I could build you something".
Walk, don't drive Walk your high street with a notepad. Look for businesses with no site or an embarrassing one. Note the name — call in the afternoon when it's quiet.
Lead with the demo, not the price Never quote a price cold. Always offer the free mockup first. Once they've seen it, they're 80% sold before price even comes up.
Post your work on Twitter Tag the business. "Built this for @[LocalCafe]" — they see it, their followers see it. Social proof + warm leads in one move.

The Twitter loop that gets 50K–300K views per post

Build → record → post. The formula that consistently generates inbound client enquiries from Twitter/X.

Step 01

Post the tools, not the pitch

Use the format: "Clo Design + Kling 3.0" and attach your screen recording. Tool-stack posts get algorithm boost from the tool companies' followers.

Step 02

CTA goes in the reply

Never put "DM me for pricing" in the main post. Wait for it to gain traction, then reply with: "I build these for clients. Starting at £[X]. Book a call →"

Step 03

Wrap in Shots.so

Upload your recording to Shots.so → pick an animated device mockup → export as MP4. Adds ~10% visual quality and makes your post stand out in a feed.

Step 04

Post daily, repost others

Repost others who recreate your prompts. Quote trending AI announcements. The algorithm rewards consistency — one post per day beats seven posts on Sunday.
